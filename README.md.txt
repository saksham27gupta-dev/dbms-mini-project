# Mobile Wallet DBMS Project

## Description
This project is a mobile wallet database system with focus on security and auditability.

## Features
- Add money
- Transfer money
- Transaction history
- Audit logging using triggers

## How to Run
1. Run schema.sql
2. Run procedures.sql
3. Run triggers.sql
4. Run sample_data.sql
5. Run demo_queries.sql

## Concepts Used
- Transactions (COMMIT, ROLLBACK)
- Stored Procedures
- Triggers
- Audit Logs
